# Plan de Estudio: Gestión de Soluciones Cloud con Google Cloud Platform

## Descripción General

Este documento describe la planificación del curso "Gestión de Soluciones Cloud con Google Cloud Platform", diseñado para el Servicio Nacional de Turismo (SERNATUR).

El curso tiene una duración total de 32 horas cronológicas, impartido en modalidad Online Sincrónica con clases teórico-prácticas.

## Objetivos Generales

Al finalizar el curso, los participantes serán capaces de aplicar las herramientas y servicios de Google Cloud Platform para diseñar, migrar y gestionar soluciones de datos en la nube, comprendiendo la arquitectura de GCP y sus herramientas principales para el análisis de datos escalable.

## Estructura del Curso

El curso se divide en 7 módulos, cada uno con objetivos de aprendizaje y contenidos específicos.

*   **Módulo 1:** Introducción a la Nube y Ecosistema GCP (4 horas)
*   **Módulo 2:** Almacenamiento y Gestión de Datos (5 horas)
*   **Módulo 3:** Migración de Datos (5 horas)
*   **Módulo 4:** Análisis de Datos con BigQuery (6 horas)
*   **Módulo 5:** Visualización de Datos (5 horas)
*   **Módulo 6:** Servicios Cognitivos y Automatización (3 horas)
*   **Módulo 7:** Proyecto Integrador (4 horas)

A continuación, se detallan los contenidos de cada módulo en sus respectivas carpetas.
