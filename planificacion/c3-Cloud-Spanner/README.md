# [Clase 3: Almacenamiento Avanzado y Migración de Datos](https://gamma.app/docs/Cloud-Spanner-Base-de-Datos-Distribuida-Globalmente-ccuil9q2gkqn9o1?mode=doc)

## Resumen de la Clase

En esta sesión de 4 horas, finalizaremos el **Módulo 2** explorando bases de datos escalables y comenzaremos con el **Módulo 3**, introduciendo las estrategias y herramientas para migrar datos a la nube de Google.

---

### **Módulo(s) Cubierto(s):**
*   **Módulo 2:** Almacenamiento y Gestión de Datos (Parte 2 de 2, 1 hora)
*   **Módulo 3:** Migración de Datos (Parte 1 de 2, 3 horas)

### **Objetivos de Aprendizaje:**
*   Evaluar el caso de uso para Cloud Spanner.
*   Comprender las diferentes estrategias de migración de datos (online/offline, homogénea/heterogénea).
*   Conocer los servicios de GCP diseñados para la migración de bases de datos y datos masivos.

### **Contenidos a Cubrir:**
1.  **Cloud Spanner (Datos estructurados - Globalmente distribuido) (1h):**
    *   Casos de uso: Cuándo elegir Spanner sobre Cloud SQL.
    *   Arquitectura y características clave (consistencia externa, escala global).
    *   Demostración de la interfaz y conceptos.
2.  **Introducción a la Migración de Datos (3h):**
    *   Conceptos de la migración de datos a la nube.
    *   **Database Migration Service (DMS):**
        *   Migraciones homogéneas a Cloud SQL.
        *   Configuración de un trabajo de migración (demostración).
    *   **Transfer Appliance / Storage Transfer Service:**
        *   Casos de uso para transferencias de datos a gran escala y online/offline.

### **Actividades Prácticas:**
*   Demostración guiada de la consola de Cloud Spanner.
*   **Taller de Migración de Datos (Parte 1):**
    *   Planificación de un escenario de migración simple.
    *   Ejercicio práctico: Mover datos desde un archivo CSV a una tabla de Cloud SQL usando herramientas de importación.
