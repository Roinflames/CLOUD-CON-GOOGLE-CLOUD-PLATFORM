# [Clase 6: De BigQuery a la Visualización de Datos](https://lookerstudio.google.com/u/0/navigation/reporting)

## Resumen de la Clase

En esta clase de 4 horas, concluiremos el **Módulo 4** con una introducción a las capacidades de Machine Learning de BigQuery y comenzaremos el **Módulo 5**, aprendiendo a transformar los datos analizados en visualizaciones y dashboards interactivos con Looker Studio.

---

### **Módulo(s) Cubierto(s):**
*   **Módulo 4:** Análisis de Datos con BigQuery (Parte 3 de 3, 2 horas)
*   **Módulo 5:** Visualización de Datos (Parte 1 de 2, 2 horas)

### **Objetivos de Aprendizaje:**
*   Conocer el potencial de BigQuery ML para crear modelos de machine learning usando SQL.
*   Comprender los principios de la visualización de datos efectiva.
*   Conectar Looker Studio a una fuente de datos como BigQuery.
*   Crear los primeros elementos de un dashboard (gráficos, tablas, filtros).

### **Contenidos a Cubrir:**
1.  **BigQuery ML (Introducción) (2h):**
    *   ¿Qué es BigQuery ML? Entrenar e implementar modelos con SQL.
    *   Tipos de modelos soportados (Regresión, Clasificación, etc.).
    *   Demostración: Crear un modelo de ejemplo (ej. `CREATE MODEL`).
    *   Evaluar y predecir con el modelo.
2.  **Introducción a la Visualización con Looker Studio (2h):**
    *   Principios básicos del diseño de dashboards.
    *   **Interfaz de Looker Studio:**
        *   Fuentes de datos (Data Sources).
        *   Informes (Reports).
    *   **Conectores:**
        *   Conectar Looker Studio a una tabla o consulta de BigQuery.
    *   **Creación de Gráficos:**
        *   Tipos de gráficos (barras, series de tiempo, circulares, etc.).
        *   Configuración de dimensiones y métricas.

### **Actividades Prácticas:**
*   **Demostración Guiada:**
    *   El relator mostrará un ejemplo de entrenamiento y predicción con BigQuery ML.
*   **Taller de Looker Studio (Parte 1):**
    *   Conectar un informe de Looker Studio a un conjunto de datos de BigQuery.
    *   Crear un primer dashboard simple con 2-3 gráficos y una tabla de datos.
